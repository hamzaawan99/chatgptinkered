# ChatGPTinkered Frontend

A React-based frontend for ChatGPTinkered, built with Vite, providing a ChatGPT-like interface for personal use.

> **Note**: This frontend was created with the help of **Vibe Coding (GitHub Copilot-powered)** since I don’t have prior frontend experience.  
> The project setup, components, and styling were generated/adapted using AI-assisted coding workflows.

## Features

- **Chat Interface**:
  - Real-time conversation
  - Markdown support with syntax highlighting
  - Code block copying functionality
  - Message rating system (thumbs up/down)
  - Token and cost display per message

- **Conversation Management**:
  - Sidebar with conversation history
  - Easy switching between conversations
  - New chat creation
  - Conversation previews

- **Settings**:
  - Custom instructions management
  - Accessible settings panel

- **Code Display**:
  - Syntax highlighting for multiple languages
  - Copy code button
  - Proper code formatting

## Technologies

- React 18
- TypeScript
- Vite
- TailwindCSS
- React Markdown
- Syntax Highlighter
- Heroicons
- **Vibe Coding (GitHub Copilot AI-assisted development)**
